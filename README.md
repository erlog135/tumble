# Tumble

A rugged watchface for Pebble, inspired by the default watchface of the first generation Garmin Instinct.



### Features

* Displays health stats, device info, and outside conditions
* Four customizable complication slots, including a miniature window and a historical graph
* Black and white theme for readability and simplicity
* Configurable seconds behavior (Always on, shown on shake, or always off)
* Compatible with every Pebble
* Given your location, uses [SunCalc](https://github.com/mourner/suncalc) for sun/moon info and [Open-Meteo](https://open-meteo.com/) for weather data
